Zadanie 0: Iloczyn wielomianów

Wprowadzenie

Jednomian zmiennej xx to wyrażenie postaci axnaxn, gdzie aa jest współczynnikiem jednomianu a nn to nieujemna liczba całkowita nazywana stopniem jednomianu. Jednomian o współczynniku równym 00 to jednomian zerowy.

Wielomianem zmiennej xx o współczynnikach a0,…,ana0,…,an nazywamy sumę jednomianów ∑ni=0aixi∑i=0naixi. Wielomian, którego wszystkie jednomiany są zerowe, to wielomian zerowy.

Stopniem niezerowego wielomianu jest maksymalny stopień jego niezerowego jednomianu. Przyjmujemy, że stopniem wielomianu zerowego jest −1−1.

Iloczyn wielomianów definiujemy wzorem

(∑i=0maixi)(∑j=0nbjxj)=∑k=0m+n(∑i+j=kaibj)xk.(∑i=0maixi)(∑j=0nbjxj)=∑k=0m+n(∑i+j=kaibj)xk.

Polecenie

Napisz program, który obliczy iloczyn dwóch wielomianów o współczynnikach całkowitych.

Postać danych

Na wejściu programu jest ciąg spacji i zapisanych dziesiętnie liczb całkowitych, dowolnie podzielony na wiersze.

Liczby

mm a0a0 …… am−1am−1 nn b0b0 …… bn−1bn−1

z nieujemnymi mm i nn, reprezentują parę wielomianów ∑m−1i=0aixi∑i=0m−1aixi oraz ∑n−1j=0bjxj∑j=0n−1bjxj.

Postać wyniku

Wynikiem programu jest jeden wiersz postaci

( uu ) ( vv ) = ww

gdzie uu i vv to wielomiany dane a ww jest ich iloczynem.

Wielomiany wypisujemy w czytelnej dla użytkownika postaci, której składnia jest zdefiniowana poniższą gramatyką w rozszerzonej notacji BNF. W gramatyce fragmenty, mogące się powtórzyć zero lub więcej razy, ujęte są w nawiasy klamrowe, fragmenty opcjonalne w nawiasy kwadratowe, symbole pomocnicze w nawiasy kątowe a symbole końcowe w cudzysłowy. Symbolem początkowym gramatyki jest <wielomian>.

<wielomian> ::= "0" | [ "-" ] <jednomian> { <operacja> <jednomian> }
<operacja> ::= "+" | "-"
<jednomian> ::= "1" | <dużo> | [ <dużo> ] "x" [ "^" <dużo> ]
<dużo> ::= "1" <cyfra> { <cyfra> } | <cyfra od 2 do 9> { <cyfra> }
<cyfra> ::= "0" | "1" | <cyfra od 2 do 9>
<cyfra od 2 do 9> ::= "2" | "3" | "4" | "5" | "6" | "7" | "8" | "9"

Dodatkowo, jednomiany wielomianów mają być wypisywane w kolejności malejących stopni.

W wierszu wynikowym nie powinno być spacji ani żadnych innych znaków, które nie zostały wymienione powyżej. Wiersz wynikowy ma być zakończony reprezentacją końca wiersza '\n'.

Przykłady

Dla danych:

6 3 0 -1 0 -4 4
2 15 -1

wynikiem programu powinno być:

(4x^5-4x^4-x^2+3)(-x+15)=-4x^6+64x^5-60x^4+x^3-15x^2-3x+45

Dla danych:

   3 2  0      1 6 -6  2 0 -1 0     0

wynikiem programu powinno być:

(x^2+2)(-x^3+2x-6)=-x^5-6x^2+4x-12

Dla danych:

       0
      5
     0
    0     
   0
  0
-3

wynikiem programu powinno być:

(0)(-3x^4)=0

Uwagi i wskazówki

  • Wolno założyć, że dane są poprawne.

  • Wolno założyć, że stopnie wielomianów danych oraz stopień ich iloczynu nie przekroczą 1010.

  • Wolno założyć, że współczynniki wielomianów danych oraz obliczenie ich iloczynu wzorem podanym powyżej nie wyjdą poza zakres typu int.

  • Program do testów będzie kompilowany poleceniem:

    gcc -std=c89 -pedantic -Wall -Wextra -Werror nazwa.c -o nazwa

    Wszystkie wymienione opcje kompilatora są obowiązkowe i nie wolno dodawać do nich żadnych innych.

  • Przyjmujemy, że wynik funkcji main() inny niż 00 informuje o błędzie wykonania programu.

  • Do treści zadania dołączone są pliki z rozszerzeniem .in zawierające przykładowe dane i odpowiadające im pliki z rozszerzeniem .out z wynikami wzorcowymi.

  • Poprawność wyniku można sprawdzić, przekierowując na wejście programu zawartość pliku .in i porównując rezultat, za pomocą programu diff, z plikiem .out, np.:

    < przyklad1.in ./zadanie0 | diff - przyklad1.out

    Wynik uznamy za poprawny tylko, jeśli jest identyczny z wynikiem wzorcowym.

  • Wyniki programu można obejrzeć w oknie przeglądarki wykonując na students polecenie

    < przyklad1.in ./zadanie0 | ( echo '# $$' ; cat ; echo '$$' ) | pandoc --standalone --mathjax -o przyklad1.html ; firefox przyklad1.html &

  • Przypominamy: punkty za to zadanie nie będą miały wpływu na końcową ocenę laboratorium.
